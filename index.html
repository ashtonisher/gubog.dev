경험한 것들
  • React · TypeScript 환경에서 프론트엔드 앱 설계, 개발, 운영 경험이 있습니다.
  • React 소스를 Next.js로 마이그레이션 개발 및 검증한 경험이 있습니다.
  • React Class 컴포넌트, ASP 등 레거시 기반 서비스를 최신 스택으로 마이그레이션한 경험이 있습니다.
  • Yarn Berry + Turbo 기반 모노레포를 직접 설계하고, 프로젝트 간 공통 패키지 구조를 구축한 경험이 있습니다.
  • 빌드 속도·번들 사이즈·LCP/CLS 등 성능 지표를 분석하고 개선한 경험이 있습니다.
  • 브라우저 호환성 이슈를 분석하고 대체 구현으로 해결한 경험이 있습니다. (iOS 17 이하 OffscreenCanvas 미지원 → CSS transition 기반 파티클 훅으로 대체)
  • 앱-웹뷰 브릿지 통신을 활용해 네이티브 메서드 호출 및 유저 컨텍스트 수신을 구현한 경험이 있습니다.
  • Android · iOS 웹뷰 환경의 이미지 렌더링, 터치 이벤트, safe-area 등 크로스 플랫폼 이슈를 다수 해결한 경험이 있습니다.
